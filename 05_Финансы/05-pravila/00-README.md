## Ключевые RBAC-роли

| Роль | Что делает в модуле |
|---|---|
| **Менеджер** | Видит свой список Order, видит маржинальность (свою). НЕ может редактировать платежи/счета. |
| **Бухгалтер** | Регистрирует платежи, выставляет счета, закрывает заказы, оформляет Refund. |
| **Директор** | Видит сводную аналитику: маржинальность, дебиторку, динамику выручки. |
| **Админ** | Настраивает справочники, управляет ролями. |
