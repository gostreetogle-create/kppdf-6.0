# 05_Финансы/02-page-orders/00-README.md — UI `/finance/orders`

> **Назначение.** UI списка заказов + детальной карточки Order: 3-зонный макет Каркас-Kit.

## 0. Контекст

Главный экран бухгалтера + директора. Менеджер видит только свои Order.

## 1. Ожидаемое содержимое

- `02-spisok-orders.md` — `/finance/orders` таблица Order: # Order, клиент, менеджер, сумма, оплачено, сальдо, статус
- `02-kartochka-order.md` — `/finance/orders/[id]` 3-зонный макет: левая панель хронология + центральная шапка+таблица счетов+таблица платежей + правая панель действия
- `02-filtri.md` — по статусу, по клиенту, по менеджеру, «просроченные счета»
- `02-knopki.md` — «Создать счёт» / «Зарегистрировать платёж» / «Закрыть заказ» / «Оформить Refund»
