## Назначение

Документирует **границы MVP** для модуля `Финансы` — что НЕ реализуется в первой версии. Источник: `МОДУЛЬ-ФИНАНСЫ.md` §7 (что в v1, а что нет).

## Что НЕ входит в v1 модуля «Финансы»

### 1. Интеграция с банком / 1С / банк-клиентом

**Что:** Автоматическая загрузка банковских выписок, сверка платежей, сверка с 1С.
**v1:** Все платежи — ручной ввод по выписке (бухгалтер вводит сумму вручную).
**v2:** Оценить ROI при 50+ платежах/месяц.

### 2. Частичные оплаты по этапам

**Что:** Предоплата 30% → производство → доплата 70% → отгрузка. Сложные схемы пропорциональных платежей.
**v1:** Только один финальный счёт или простой аванс (50/50). Авто-FIFO отложено (СПОР-7: ручной выбор в v1).
**v2:** При доминировании долгосрочных контрактов.

### 3. Авто-перевыставление счетов при просрочке

**Что:** Автоматическое создание нового Invoice при просрочке предыдущего.
**v1:** Только красная подсветка просроченных счетов, менеджер сам решает.
**v2:** При росте объёмов дебиторки.

### 4. Финансовая аналитика (NPV, графики, прогнозы)

**Что:** NPV, графики помесячно, прогноз поступлений, когортный анализ.
**v1:** Только списки и простые сводки (дебиторка по клиентам, маржа по менеджерам).
**v2:** При запросах руководства.

### 5. Мультивалюта (USD/EUR/KZT)

**Что:** Работа с несколькими валютами, конвертация по курсу, multi-currency отчёты.
**v1:** **Только RUB** жёстко (СПОР-14: `currency = 'RUB'` default). UI не показывает dropdown выбора валюты.
**v2:** При выходе на иностранных клиентов.

### 6. Автоматическое формирование актов (сверки)

**Что:** Автоформирование актов сверки по завершённым Order.
**v1:** Акты — отдельный процесс вручную.
**v2:** При запросах клиентов / налоговых.

### 7. Мобильная версия финансовых экранов

**Что:** Адаптив для мобильных устройств с полным функционалом.
**v1:** Нет. Desktop-first.
**v2:** После основного desktop-релиза.

## Принцип: «минимум в первой версии, реальная потребность в v2»

Все пункты выше могут быть пересмотрены при **реальных запросах пользователей**. Если топ-5 клиентов начнут массово просить авто-FIFO — перевод в v1.1.
