# 05-zhiznennyj-cikl/ — Точка входа подпапки жизненного цикла модуля «Финансы»

> **Назначение.** Оглавление подпапки `05-zhiznennyj-cikl/` модуля Финансы — содержит статусы (Order, Invoice, Payment) и налоговый учёт (УСН/НДС). Источник: `МОДУЛЬ-ФИНАНСЫ.md` §3 «Жизненный цикл» + §6 «Бизнес-правила».

> ⚠️ **STUB** (создан Архитектором 2026-06-27, ТЗ-009). Будет заполнен Бизнес-аналитиком в Run 5 Аналитика (ТЗ-010).

## Содержание

| Файл | Назначение |
|---|---|
| `05-statusy.md` | Статусы: Order (5), Invoice (5), Payment (2 типа) |
| `05-nalogovyi-uchet.md` | Налоговый учёт: УСН + НДС (1 ставка 20%) + маржа расчёт |

## Ключевые связи

- **Order → Contract:** `Contract.status='SIGNED'` → авто-создание Order (СПОР-5).
- **Order → ProductionOrder:** `ProductionOrder.status='DONE'` → Order в `AWAITING_PAYMENT`.
- **Order → Shipment:** `Shipment.status='SHIPPED'` → себестоимость для маржи.
- **Invoice → Payment:** ручной выбор через dropdown (СПОР-7).
- **Refund → Order:** полный возврат при отменённом ЗК → `Order.status='CANCELLED'`.
