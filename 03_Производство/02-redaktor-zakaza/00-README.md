> **Назначение.** Папка для UI: доска ЗК + детали ЗК + карточка задачи + buttons.

## 0. Контекст

Главный экран начальника производства (Kanban) + детальная карточка конкретного ЗК + карточка конкретной задачи.

## 1. Ожидаемое содержимое

- `02-doska-zakazov.md` — `/production` Kanban: все активные ЗК с цветовой разметкой по статусам и дедлайнам
- `02-detali-zakaza.md` — `/production/[id]` 3-зонный макет Каркас-Kit
- `02-kartochka-zadachi.md` — `/production/[orderId]/task/[taskId]`: кнопки «Начать / Частично / Готово / Сообщить о проблеме»
- `02-knopki.md` — кнопки по статусам ЗК (Создан / Планируется / В работе / Завершён / Отменён)
