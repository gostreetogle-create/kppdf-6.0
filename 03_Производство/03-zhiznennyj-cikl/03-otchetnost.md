# 03_Производство/03-zhiznennyj-cikl/03-otchetnost.md — Отчётность модуля «Производство»

> ⚠️ **STUB.** Создан декомпозицией модуля `03_Производство/` (см. [MODULE-DECOMPOSITION-PLAN.md §3](../../99_Справочники/MODULE-DECOMPOSITION-PLAN.md)).
> Контент будет наполнен Аналитиком Run 3 (ТЗ-008).

## Назначение

Отчётность для начальника производства: сводка по заказам, прогресс, просроченные, загрузка цехов. Из [`МОДУЛЬ-ПРОИЗВОДСТВО.md`](../МОДУЛЬ-ПРОИЗВОДСТВО.md) §8.

## Контекст

Из [`МОДУЛЬ-ПРОИЗВОДСТВО.md`](../МОДУЛЬ-ПРОИЗВОДСТВО.md) §8 + §4.1: начальник производства видит доску с фильтрами (по статусу, цеху, ответственному, дате, клиенту, «Только просроченные»). Колонки: Номер, Клиент, КП-источник, Договор, Дата создания, План. завершения, Статус, Прогресс, Ответственные.

## План наполнения (для Аналитика Run 3)

- Доска ЗК (`/production`): таблица/Канбан со всеми активными заказами.
- Фильтры: статус, цех, ответственный, дата, клиент, просроченные.
- Колонки: Номер, Клиент, КП-источник, Договор, даты, статус, прогресс, ответственные.
- Аналитика по цехам: загрузка, средний срок выполнения.
- MVP boundary: диаграмма Ганта — НЕ в первой версии.
