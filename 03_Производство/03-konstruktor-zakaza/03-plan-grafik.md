# 03_Производство/03-konstruktor-zakaza/03-plan-grafik.md — План-график работ

> ⚠️ **STUB.** Создан декомпозицией модуля `03_Производство/` (см. [MODULE-DECOMPOSITION-PLAN.md §3](../../99_Справочники/MODULE-DECOMPOSITION-PLAN.md)).
> Контент будет наполнен Аналитиком Run 3 (ТЗ-008).

## Назначение

План-график распределения задач по цехам с датами. Начальник производства при статусе «Планируется» распределяет: какая задача → в какой цех → ответственный → срок.

## Контекст

Из [`МОДУЛЬ-ПРОИЗВОДСТВО.md`](../МОДУЛЬ-ПРОИЗВОДСТВО.md) §3 (Ключевой сценарий, шаг 5): начальник распределяет задачи по цехам с плановыми датами. Поля `plannedStartDate` / `plannedEndDate` на ЗК, `plannedStartDate` / `plannedEndDate` на каждой задаче. Diаграмма Ганта — НЕ в MVP (§9).

## План наполнения (для Аналитика Run 3)

- Механика распределения: начальник назначает цех + ответственного + даты.
- Автоматическое заполнение `actualStartDate` при первом «В работе».
- Автоматическое заполнение `actualEndDate` при «Завершён».
- Фильтр «Только просроченные»: план. дата прошла, статус ≠ завершён.
- Проверка: дата окончания ≥ дата начала.
