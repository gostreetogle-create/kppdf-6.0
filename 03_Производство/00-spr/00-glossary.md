## Назначение

Глоссарий терминов модуля `Производство` — единый источник для всех downstream-документов. Все термины должны совпадать с [`GLOSSARY-MASTER.md`](../../99_Справочники/GLOSSARY-MASTER.md).

## Термины (8 ключевых, из `МОДУЛЬ-ПРОИЗВОДСТВО.md` §0)

| Термин | Что значит |
|---|---|
| **Производственный заказ (ProductionOrder)** | Внутренний документ на изготовление. Уникальный номер **ЗК-XXXX**. Создаётся автоматически при оплате КП. |
| **Задача (ProductionTask)** | Атомарная часть ЗК: «выплавить 100 деталей», «собрать корпус», «монтаж». Одна ЗК → несколько Задач. |
| **Цех (Workshop)** | Место выполнения задачи. Фиксированный список: «Металлоконструкции», «Сборочный цех», «Выездная бригада» и др. |
| **Ответственный** | Сотрудник (User с ролью `production`), назначенный на задачу. |
| **Частичная готовность** | Задача выполнена «частично» (50 из 100 деталей). Документируется в `quantityActual`. |
| **Parent КП (parentProposalId)** | Ссылка на КП, при оплате которого создан ЗК. Строгий FK, не NULL. |
| **Parent Договор (parentContractId)** | Ссылка на Договор (если КП был сконвертирован). Не строгий FK, может быть NULL. |
| **ЗК (Заказ-Картонка / ProductionOrder)** | Сокращение для «Производственный заказ». Номер ЗК-XXXX, счётчик отдельный от КП (СПОР-13). |

## Типы статусов (cross-ref)

Полный список статусов — в [`../03-zhiznennyj-cikl/03-statusy.md`](../03-zhiznennyj-cikl/03-statusy.md). Краткая сводка:

- `CREATED` (Создан)
- `PLANNED` (Планируется)
- `WAITING_MATERIALS` (Ожидает материалов)
- `IN_PROGRESS` (В работе)
- `PARTIAL` (Часть готова)
- `COMPLETED` (Завершён)
- `CANCELLED` (Отменён)
- `ARCHIVED` (Архив)

## Принципы наполнения (Аналитик Run 3)

1. **Один термин — одно определение.** Если термин пересекается с КП/Договором/Складом — дать cross-ref, не дублировать.
2. **Snapshot-семантика:** позиции копируются из КП как snapshot при создании ЗК. Изменение справочника товаров НЕ влияет задним числом.
3. **Связь с общим глоссарием:** новые термины → добавлять **сюда** + **в GLOSSARY-MASTER.md**.
