# 03_Производство/00-spr/00-resources.md — Ресурсы модуля «Производство»

> ⚠️ **STUB.** Создан декомпозицией модуля `03_Производство/` (см. [MODULE-DECOMPOSITION-PLAN.md §3](../../99_Справочники/MODULE-DECOMPOSITION-PLAN.md)).
> Контент будет наполнен Аналитиком Run 3 (ТЗ-008).

## Назначение

Описание ресурсов производства: оборудование, люди, материалы. Уникальный для Производства файл (нет аналога в КП/Договоре).

## Контекст

Из [`МОДУЛЬ-ПРОИЗВОДСТВО.md`](../МОДУЛЬ-ПРОИЗВОДСТВО.md) §3: ресурсы — это то, что нужно для выполнения задач. Включает оборудование цехов, людей (ответственных) и материалы (комплектующие). Связь со Складом — при нехватке материалов ЗК получает статус «Ожидает материалов».

## План наполнения (для Аналитика Run 3)

- Типы ресурсов: `EQUIPMENT` (оборудование), `HUMAN` (люди), `MATERIAL` (материалы/комплектующие).
- Связь ресурсов с цехами: какое оборудование в каком цехе.
- Связь с Складом: `StockRecord.quantity` ↔ требуемое количество в ЗК.
- Проверка доступности: при распределении задач — достаточно ли ресурсов.
- MVP boundary: MRP (автоматический расчёт потребности) — НЕ в первой версии.
