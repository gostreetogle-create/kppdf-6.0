# 02_Договор/02-redaktor-dogovora/00-README.md — UI Редактор договора

> **Назначение.** Папка для экранов и механик редактора Договора: конвертация из КП, спецификация, кнопки управления, версионирование.

## 0. Контекст

Главный пользовательский поток: КП отправлен → клиент согласился → менеджер жмёт «Конвертировать в договор» → открывается редактор Договора.

## 1. Ожидаемое содержимое (для будущего Архитектора → UX)

- `02-konvertaciya.md` — механика «Конвертировать из КП» (snapshot КП + присвоение Д-XXXX)
- `02-spetsifikatsiya.md` — UI позиций Договора (`ContractItem`): рамочный/обычный режимы
- `02-versiya.md` — кнопка «Создать новую версию» (`parentContractId`)
- `02-pechat-pdf.md` — формирование PDF по выбранному `DocumentTemplate`
- `02-knopki.md` — кнопки по статусам: Отправить клиенту / Подписать / Архивировать / Расторгнуть
