## Назначение

Папка **ограничений** модуля «Договор» — RBAC (роли → действия) и инварианты (бизнес-правила, нарушение которых = ошибка). Это **то, что защищает систему** от некорректных операций.

## Принципы наполнения (Аналитик Run 2)

1. **RBAC mirror КП + расширения:** начать с [`../../01_КП/04-pravila/04-rbac.md`](../../01_КП/04-pravila/04-rbac.md) и добавить Договор-специфичные действия (Подписать, Архивировать, Создать новую версию).
2. **Инварианты:** каждое правило = (предикат + сообщение ошибки + куда ведёт нарушение). Например:
   - «Договор нельзя создать из Proposal.status = PAID» → ошибка 422 + red toast.
