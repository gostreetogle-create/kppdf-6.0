## 2. Направление ответов для Аналитика

| OQ | Рекомендуемая гипотеза (не финальная — обоснование в Run1 Аналитика) |
|---|---|
| OQ-001 | 1 Договор = 1 ЗК в v1 (V-026 ✅ зафиксировано; multi-ЗК → v2) |
| OQ-002 | Нельзя редактировать после `SENT`. Snapshot полей уже заблокирован. |
| OQ-003 | КП остаётся в `CONVERTED` навсегда (V-026 ✅ зафиксировано, аналитика через JOIN по `packageTag`) |
| OQ-004 | Новая версия через `parentContractId` (V-026 ✅, без смены customerId для старого документа) |
| OQ-005 | Вероятно ТОЛЬКО director (повышенные права — аналог отмены ЗК) |
