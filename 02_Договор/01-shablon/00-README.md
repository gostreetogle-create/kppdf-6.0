## Ожидаемые шаблоны v1 (из `МОДУЛЬ-ДОГОВОР.md` §4.3)

### 1. «Договор поставки со Спецификацией» (default)

- Заголовок: «ДОГОВОР ПОСТАВКИ № Д-XXXX»
- Подписанты: наша организация + клиент
- Спецификация — приложение (отдельная страница или прицеплена)
- Срок действия: явный или бессрочный
- Условия оплаты: по этапам / единым платежом

### 2. «Рамочный договор»

- Заголовок: «РАМОЧНЫЙ ДОГОВОР № Д-R-XXXX»
- Подписанты: те же
- Спецификация — БЕЗ цен (только номенклатура + количество)
- Условия: цены согласовываются по факту поставки
