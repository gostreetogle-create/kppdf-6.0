## Контракты с другими модулями

| Контракт | Trigger | Target |
|---|---|---|
| **КП → Договор** | КП переходит в «PAID» (Run 1 КП) | Создаётся Договор с реквизитами КП |
| **Договор → Производство** | Договор переходит в «ACTIVE» | Создаётся Заказ на производство |
| **Договор → Финансы** | Договор подписан + платежи пошли | Учёт движения денег |
