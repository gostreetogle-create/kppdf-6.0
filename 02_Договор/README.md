## 0. Контекст

Договор — юридическое закрепление договорённости из КП. Префикс `Д-XXXX`. Создаётся конвертацией из КП в статусах `SENT` или `ACCEPTED` (КАТЕГОРИЧЕСКИ запрещено из `PAID` — V-024 ✅ зафиксировано). Содержит 7 статусов: `DRAFT → SENT → SIGNED → IN_PROGRESS → COMPLETED → ARCHIVED` + `TERMINATED` (добавлено 24.06.2026). Типы договора: «Договор поставки со Спецификацией» (по умолчанию) и «Рамочный» (без цен в спецификации).

5 папок модуля (per AGENT-METHOD.md §3 Правило 3.1): `00-spr/` (справочники) → `01-shablon/` (шаблоны) → `02-redaktor-dogovora/` (UI конструктора) → `03-zhiznennyj-cikl/` (жизненный цикл) → `04-pravila/` (ограничения).
