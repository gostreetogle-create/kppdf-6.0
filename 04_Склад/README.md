# 04_Склад/ — Модуль Склад

> **Назначение.** Документация модуля «Склад». Операционная точка контроля движения товара: приход от поставщиков, отгрузка клиентам, списание брака, перемещения между складами, резервирование под КП/ЗК, инвентаризация.

## Структура модуля (после декомпозиции)

```
04_Склад/
├── README.md                          # ← этот файл (entrypoint)
├── 00-spr/                            # Справочники (фундамент)
├── 04-konstruktor-dvizhenia/          # Движения товаров (приход/расход/перемещение)
├── 04-zhiznennyj-cikl/               # Процесс (статусы, резервы, инвентаризация)
└── 04-pravila/                        # Ограничения (RBAC + инварианты)
```

> **Цель для Run 4 Аналитика:** заполнить 19 STUB правилами (RBAC ≥30 + biznes-pravila ≥25 + statusy 4 документов + prihod/rashod/peremeshenie specs +rezervy/inventarizacia).
