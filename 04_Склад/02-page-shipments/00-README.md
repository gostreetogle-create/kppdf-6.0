# 04_Склад/02-page-shipments/00-README.md — UI `/shipments`

> **Назначение.** UI списка отгрузок ОТК-XXXX с вкладками и авто-предложением создать ОТК при завершении ЗК.

## 0. Контекст

Кладовщик отгружает товар клиенту. Страница получает авто-сигналы от Модуля Производство (ЗК COMPLETED для ITEM).

## 1. Ожидаемое содержимое

- `02-spisok-otk.md` — таблица Shipment: # ОТК, клиент, что отгружается, статус
- `02-tab-planovye.md` — ОТК в статусах `planned / packed`
- `02-tab-v-puti.md` — ОТК в статусах `shipped`, ожидание доставки
- `02-knopka-otpravit.md` — «Отгрузить» (переход в `shipped`, авто StockMovement:OUT + OrderClosing.progress++)
