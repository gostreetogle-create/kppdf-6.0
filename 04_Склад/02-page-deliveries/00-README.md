# 04_Склад/02-page-deliveries/00-README.md — UI `/supplier-deliveries`

> **Назначение.** UI списка приходов СД-XXXX с вкладками «Ожидаемые» / «Завершённые» / «Отменённые».

## 0. Контекст

Кладовщик принимает товар от поставщиков, оформляет СД. Здесь же видит ожидаемые приходы (по ЗП).

## 1. Ожидаемое содержимое

- `02-spisok-sd.md` — таблица SupplierDelivery: # СД, поставщик, дата, статус, итого
- `02-tab-ozhidaemye.md` — СД в статусах `planned / in_transit`
- `02-tab-zavershyonnye.md` — СД в статусах `completed / partially_received`
- `02-knopka-prinyat.md` — «Принять приход» (переход в `received` → `completed`, авто StockMovement:IN)
