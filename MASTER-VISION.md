## 1. VISION (1 параграф)

**KPPDF CRM v6** — специализированная **internal CRM-система** для управления полным коммерческим циклом производственной компании: от первого КП клиенту до маржинального отчёта. Пять модулей покрывают цепочку «**КП → Договор → Производство → Склад → Финансы**» в **одной** системе с **32 сущностями БД**, **7 ролями**, **38 согласованными бизнес-решениями**, **15 закрытыми спорами** и **73/73 пройденными V-проверками**. Phase 0 (теория) **100% завершена**, Phase 1 Bootstrap (Prisma миграции) разблокирован, Phase 2 UI стартует после аудита (ТЗП-001).

> ⚠️ **SCOPE-GUARDS (НЕ НАРУШАТЬ):** single-tenant / **≤10 человек в команде** / **полуавтомат** / **НЕ scaleable** для external SaaS. Полный список anti-features + UX-принципы — в [`99_Справочники/BUSINESS-VISION.md`](99_Справочники/BUSINESS-VISION.md).
