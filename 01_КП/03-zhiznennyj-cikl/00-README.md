# 03-zhiznennyj-cikl/ — Жизненный цикл КП (процесс)

> **Назначение.** Папка для описания того, как КП движется по статусам, какие переходы возможны, что происходит при конвертации в Договор.
>
> **Контекст.** Состояния КП: Черновик → Новая версия → Отправлено → Принято → Оплачено → Конвертировано → Архив → Отменено (8 статусов). При «Оплачено» авто-создаётся Производственный заказ.
>
> **Создано как:** ⚠️ STUB. Контент будет наполнен Бизнес-аналитиком (state-машина) + Архитектором (распределение файлов).
