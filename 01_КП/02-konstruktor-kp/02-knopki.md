# 02-knopki.md — Все кнопки управления КП (по статусам)

> **Назначение.** Спецификация 13 кнопок редактора КП с указанием видимости/активности по статусам: Сохранить / Отправить (SENT, в «SENT» повторно меняет адрес) / Отметить как принятое (отменяемо до Оплачено или Конвертировано) / Отметить как отклонённое / Отметить как оплаченное (авто-Order ЗК-XXXX) / Конвертировать в договор (выбор шаблона в мини-форме) / Создать новую версию (линейная цепочка) / Скопировать для другого юрлица / Печать PDF / Архивировать (только DRAFT и Отклонено) / Восстановить (из архива, возврат в прежний статус) / Добавить товар / Удалить позицию.

## 0. Контекст

Синонимы кнопки «Создать новую версию» — «Склонировать» / «Сделать копию» — это естественная речь менеджера, в UI каноническое имя одно. Кнопки «Скопировать для другого юрлица» нет в статусе Отклонено (смысла нет — клиент отказался) и Конвертировано (есть договор, редактируйте его).
