# 02-konstruktor-kp/ — Сборка КП (UI + механика)

> **Назначение.** Папка для экранов и механик редактора КП: витрина + корзина (Шаг 1), 3-зонный редактор (Шаг 2), таблица позиций, killer-features (смена юрлица, автосохранение), кнопки управления.
>
> **Контекст.** Главный пользовательский поток: менеджер открывает карточку клиента → витрина товаров → добавляет в корзину → переходит в редактор → формирует КП → отправляет. Всё это собирается здесь.
>
> **Создано как:** ⚠️ STUB (`§4.5 AGENT-METHOD.md Правило 4.5.1`). Содержимое будет наполнено Архитектором + UX-проектировщиком + Тех.писателем.

---

## 0. Контекст

Это «центральная» папка модуля КП — здесь сходятся все остальные части (шаблоны, ЖЦ, правила). Большинство killer-features проекта живёт здесь (смена юрлица за 30 сек, 3-зонный редактор на Каркас-Kit, автосохранение с debounce).

## 1. Ожидаемое содержимое (для будущего Архитектора)

- `02-korzina.md` — витрина товаров + корзина (Шаг 1 цикла).
- `02-redaktor.md` — 3-зонный редактор: левая панель / центр A4 / правая панель.
- `02-tablica-pozicij.md` — математика: наценки × скидки × НДС, скрытые столбцы «Скидка %», «Применить общую скидку».
- `02-smena-yurlitsa.md` — killer-feature «30 секунд» (пересчёт цен по наценке юрлица).
- `02-avtosoxranenie.md` — debounce 5-10 сек + localStorage + optimistic locking.
- `02-knopki.md` — все кнопки (Сохранить, Отправить, Создать новую версию, Скопировать для другого юрлица и т.д.) по статусам.
