# 02-tablica-pozicij.md — Таблица позиций КП

> **Назначение.** Спецификация таблицы позиций в превью A4: 13 колонок (№ / Фото / Артикул / Название / Тип / Единица / Количество / Цена / Наценка% / Скидка% / Итого / Действие), формула итога (`итого = кол-во × цена × (1 + наценка) × (1 − скидка)`), пересчёт итога при изменении «Итого» вручную (система вычисляет нужный % скидки). Скрытые столбцы «Скидка %» и «Себестоимость» (только для экрана, по чекбоксам). Две механики добавления товара: ОСНОВНАЯ — инлайн-поиск в пустой строке таблицы; ДОПОЛНИТЕЛЬНАЯ — витрина слева.

## 0. Контекст

Правило приоритета: общая скидка КП применяется ко всем позициям автоматически; индивидуальная скидка в ячейке перекрывает общую для этой позиции. Пометка «общ.» в скрытом столбце «Скидка» показывает, что это унаследованная общая скидка, а не индивидуальная.
