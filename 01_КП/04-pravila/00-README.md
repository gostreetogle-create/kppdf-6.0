# 04-pravila/ — Ограничения и права (RBAC + бизнес-правила)

> **Назначение.** Папка для сводной RBAC-матрицы модуля КП + всех бизнес-правил (инварианты, цифровые ограничения, политики).
>
> **Контекст.** Финальная папка в иерархии 00→01→02→03→04. Здесь собираются ВСЕ ограничения модуля в явном виде, чтобы ни в каком другом файле правила не нарушались «втихую».
>
> **Создано как:** ⚠️ STUB. Контент — Бизнес-аналитик (правила) + возможно RBAC-аудит.

---

## 0. Контекст

Эта папка — «защитный слой» модуля. Даже если в предыдущих папках ошибка, здесь должно быть достаточно правил, чтобы поймать её через QA-валидатора.

## 1. Ожидаемое содержимое (для будущего Архитектора)

- `04-rbac.md` — RBAC-таблица для КП: кто может переводить между статусами, кто видит себестоимость, кто может удалять/архивировать. Ссылается на `99_Справочники/RBAC-MATRIX.md`.
- `04-biznes-pravila.md` — все инварианты КП: `цена ≥ 0`, `скидка ≤ 100%`, `минимальная маржа`, **правило одной ставки НДС**, запрет конвертации оплаченного КП.

## 2. Связанные документы

- [`../../AGENT-ROLES.md` §2.2](../../AGENT-ROLES.md) — Бизнес-аналитик (правила и инварианты — это его зона)
- [`../../99_Справочники/RBAC-MATRIX.md`](../../99_Справочники/RBAC-MATRIX.md) — общая RBAC-матрица проекта
- [`../../99_Справочники/СПОРНЫЕ-МОМЕНТЫ.md`](../../99_Справочники/СПОРНЫЕ-МОМЕНТЫ.md) — 15 зафиксированных дискуссий, некоторые из которых касаются КП
