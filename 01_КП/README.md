## 0. Контекст

КП — первый документ в цикле продажи (`КП → Договор → ЗК → Склад → Финансы`). Модуль решает три ключевые боли в сравнении с v5.0: копирование КП для другого юрлица (15 мин → 30 сек), долгая корзина (инлайн-поиск), PDF с логотипом и печатью.
