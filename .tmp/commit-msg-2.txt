chore(gitignore): untrack tsconfig.tsbuildinfo + next-env.d.ts

Both files are auto-regenerated by tsc / next dev on every build and were
causing rebase friction (unstaged modifications blocking git pull --rebase).
Untracking them keeps the working tree clean for future rebase operations.

- tsconfig.tsbuildinfo: tsc incremental build cache
- next-env.d.ts: next.js auto-generated types
