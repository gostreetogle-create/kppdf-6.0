# AGENT-REVIEW.md — Чек-лист самопроверки

> **Назначение.** Прежде чем сказать «готово», пройди по всем пунктам этого списка. Пункты MUST = блокеры сдачи. Пункты SHOULD = рекомендации.
>
> **Обязательное условие:** до ознакомления с этим файлом агент ОБЯЗАН прочитать [`AGENT-ENTRYPOINT.md`](AGENT-ENTRYPOINT.md) и [`AGENT-FORMAT.md`](AGENT-FORMAT.md).

---

## 0. Контекст

Критерии разделены на **два уровня**:
- **🔴 MUST** — провал = не сдавать. Исправить без исключений.
- **🟡 SHOULD** — рекомендация. Допустимо отметить с обоснованием (например, файл-заглушка STUB).

---

## 1. Структура и форматирование

- [ ] Файл в правильной папке (соответствует идеальной структуре модуля)?
- [ ] Заголовок `#` явно говорит, что это за файл (одно предложение)?
- [ ] Есть блок `> **Назначение.**` в начале (1-2 строки)?
- [ ] Разделов не больше 6-7? Если больше — дробление или объединение
- [ ] Нет заголовков глубже `###` без обоснования?
- [ ] **🔴 MUST** Объём файла 80-250 строк (target); > 400 (hard limit) — обязательное дробление?

---

## 2. Содержание

- [ ] В начале раздела явно: что входит И что НЕ входит?
- [ ] Каждое правило с номером (`Правило 1.`, `Правило 2.`)?
- [ ] Каждое поле имеет тип + описание + источник?
- [ ] Каждая таблица имеет шапку?
- [ ] Каждое утверждение — правило/поле/переход/таблица/сценарий?
- [ ] Нет «вероятно», «обычно», «может быть», «бесила менеджеров»?
- [ ] Нет вводных абзацев длиннее 3 строк без конкретики?

---

## 3. Связи и кросс-ссылки

- [ ] Каждая кросс-ссылка указывает на существующий файл?
- [ ] Каждая кросс-ссылка указывает на существующее поле/раздел?
- [ ] Указано, откуда берутся данные для каждого поля?
- [ ] Все термины или в `GLOSSARY-MASTER.md` или в локальном глоссарии?
- [ ] Нет дублирования информации между файлами?

---

## 4. Самодостаточность

- [ ] Можно ли прочитать файл отдельно и понять о чём он?
- [ ] Все зависимости от других файлов ЯВНО указаны?
- [ ] Программист, читая только этот файл, знает ЧТО делать?

---

## 5. Edge-кейсы

- [ ] Раздел «Edge-кейсы» присутствует?
- [ ] Покрыты минимум 3 типичных сценария: «нормальный», «ошибочный», «параллельный»?
